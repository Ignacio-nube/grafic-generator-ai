# Graficos AI

Generador de gráficos inteligente impulsado por IA.

## Características

- Generación de gráficos a partir de texto natural
- Soporte para gráficos de barras, líneas, pastel y área
- Descarga de gráficos en PDF
- Modo oscuro/claro
- Fuentes y descripciones automáticas

## Tecnologías

- React
- TypeScript
- Vite
- Chakra UI
- Recharts
- OpenAI API
